# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ts-hooks-kit, please report it responsibly using GitHub Security Advisories instead of opening a public issue.

**Report here:** https://github.com/naufaldi/ts-hooks-kit/security/advisories

Please do not disclose security vulnerabilities publicly. The maintainers will acknowledge your report within 7 days and work with you on a fix.

Thank you for helping keep ts-hooks-kit secure.
