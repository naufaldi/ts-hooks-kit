Manage a `Set` data structure with reactive updates.

Returns a tuple of `[set, actions]` where actions include:

- `add(value)` — add a value to the set
- `remove(value)` — remove a value from the set
- `toggle(value)` — add or remove a value
- `has(value)` — check if a value exists
- `clear()` — empty the set
- `reset()` — restore the initial set
